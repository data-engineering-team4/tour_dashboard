# ✈️ 국내 여행 어디로 갈까?
### tour API를 활용한 국내 관광 정보 시각화 대시보드
<br>

## 프로젝트 개요
### 목적
공공 데이터 포털의 관광 관련 api와 스마트 치안 빅데이터 플랫폼의 지역별 체감 안전도 api를 활용해 편의 시설, 행사 정보, 많이 가는 지역 등 국내 관광의 다양한 정보를 시각화하여 제공한다.
